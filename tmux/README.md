## Installation

- Install [tmux-plugins/tpm](https://github.com/tmux-plugins/tpm).

  ```bash
  git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
  ```

- Source my setting files in `~.tmux.conf`.

  ```tmux
  source [path_to_dotfiles/tmux/tmux.conf]
  ```
